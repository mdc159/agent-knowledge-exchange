# Paperclip Knowledge

Paperclip notes are split by operational concern:

- [Company Setup](paperclip-company-setup.md): company/runtime wiring, adapter
  choices, and company-scoped runtime state.
- [Access and CLI Auth](paperclip-access-and-cli-auth.md): board access, CLI
  auth, headless VPS auth flow, and access troubleshooting.

Use the company setup note when creating or wiring a company. Use the access
and auth note when a deployment is up but you still cannot administer it.
