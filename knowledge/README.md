# Knowledge Index

This directory contains durable, reusable operating knowledge. Treat live
addresses, service status, inventory, and handoff claims as time-sensitive and
verify them against the current system before acting.

## Domains

- [Hermes](hermes/README.md): Hermes runtime operations, memory, agent coordination,
  and sanitized recovery material
- [Architecture](architecture/empire-topology.md): system models and documented boundaries
- [Deployment](deployment/reference-node-bootstrap.md): reusable node bootstrap procedures
- [Open WebUI](open-webui/README.md): Open WebUI operating notes and integrations
- [Paperclip](paperclip/README.md): Paperclip setup and access guidance
- [Research](research/README.md): curated engineering and agent-system research
- [Tijuana](tijuana/): regional business, registration, and operating knowledge
- [n8n](n8n/README.md): n8n workflow notes

## Content Standard

Add material only when another operator or agent can reuse, audit, or restore
from it. Keep changing runtime state in its authoritative system and summarize
only the durable lesson here. See the [repository policy](../docs/repo-policy.md)
and [contribution guide](../docs/contribution-guide.md) before adding content.
