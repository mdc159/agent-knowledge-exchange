# Company Memory Seeding

## Purpose

Seed a new company with curated lessons without contaminating it with ambient
memory from another company.

## Procedure

1. start the company with a clean runtime home
2. identify specific lessons worth transferring
3. summarize those lessons into explicit artifacts
4. import only the approved summaries or skills
5. note the transfer in the company backup or profile docs

## Rule

Cross-company memory transfer is explicit, not ambient.
